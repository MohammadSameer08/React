# React Learning Projects

A collection of React projects exploring different approaches and tools.

## Projects

| Project | Description | Setup |
|---------|-------------|-------|
| **01basicreact** | React with Create React App | `npm start` |
| **01basicvite** | React with Vite bundler | `npm run dev` |
| **customReact** | Custom React implementation from scratch | Open `index.html` in browser |
| **02counter** | Counter app (Vite + Hooks) | `npm run dev` |

## Quick Start

Each project is independent. Navigate to any folder and run the command from the Setup column above.

```bash
cd <project-folder>
npm install  # if needed
npm run dev  # or npm start for basicreact
```

## Topics Covered

- React fundamentals & components
- Vite vs Create React App
- React Hooks & state management
- Custom React rendering
